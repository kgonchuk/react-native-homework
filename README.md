# 📍 PinPointMe — Мобільний додаток для обміну локаціями та фотографіями

**PinPointMe** — це сучасний Fullstack мобільний додаток на базі React Native (Expo), створений для того, щоб користувачі могли фіксувати свої улюблені подорожі чи локації, додавати фотографії, залишати коментарі, ставити лайки та переглядати мітки інших користувачів на інтерактивній карті.

## 🛠️ Технологічний стек

### Frontend (Мобільний додаток):
* **React Native & Expo** (Expo Router для навігації, Expo ImagePicker для роботи з галереєю)
* **Redux Toolkit** (Керування глобальним станом додатку, асинхронні Thunk-операції для запитів)
* **React Native Maps** (Відображення маркерів та інтерактивних карт)
* **Vector Icons & StyleSheet** (Сучасний та адаптивний UI/UX дизайн)

### Backend (Серверна частина):
* **Node.js & Express** (Побудова надійної архітектури REST API)
* **MongoDB Atlas & Mongoose** (Хмарне збереження даних користувачів, постів та коментарів)
* **Cloudinary & Multer Cloud Storage** (Постійне та безпечне збереження зображень у хмарі)
* **JWT (JSON Web Tokens) & Bcrypt** (Надійна автентифікація, безпека сесій та шифрування паролів)
* **Render** (Хостинг та деплой серверного коду в режимі Live)

---

## 🚀 Основний функціонал

* 🔐 **Повна автентифікація:** Реєстрація користувачів із завантаженням унікального аватара на Cloudinary, безпечний логін, логаут та механізм оновлення токенів (Access/Refresh JWT).
* 📸 **Створення публікацій:** Можливість додавати нові пости із назвою, описом місцевості, фотографією та точними геокоординатами.
* 🗺️ **Інтерактивна карта:** Перегляд точного розташування кожного поста на карті за допомогою вбудованого модуля навігації.
* 💬 **Соціальна взаємодія:** Можливість ставити/прибирати лайки іншим користувачам та додавати коментарі до постів у реальному часі.
* 👤 **Особистий профіль:** Відображення аватара користувача та автоматично відфільтрована стрічка, яка містить публікації виключно цього автора.
